VMware Fusion虚拟机避开虚拟机检测

发表于 2019-11-28 更新于 2019-11-29 分类于 MacBook
本文字数： 453 阅读时长 ≈ 1 分钟

部分软件启动后会检测其是否运行在虚拟机中,如果是则退出，下面给出方案避开虚拟机检测。

在虚拟机未启动的情况下（如果虚拟机开着不能保存配置）打开 Finder 在上边的搜索框里输入 VMX 就会搜索到一个以.vmx 结尾的文件

这个文件就是虚拟机的配置文件，如果有多个虚拟机请根据你想配置的虚拟机名字来配置。

目录因人而异，注意将anchor替换为自己的用户名，我的目录是：

/Users/anchor/Documents/VirtualMachines.localized/Windows7x64.vmwarevm/Windows7x64.vmx

之后用文本编辑器打开这个配置文件，在末尾加上：

monitor_control.restrict_backdoor = "TRUE"
isolation.tools.setVersion.disable = "TRUE"
isolation.tools.getVersion.disable = "TRUE"

保存文件，启动虚拟机。搞定。
